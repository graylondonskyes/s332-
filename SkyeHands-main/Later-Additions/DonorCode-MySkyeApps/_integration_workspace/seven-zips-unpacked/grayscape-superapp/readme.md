# GrayScape // THE NEXUS — Installable App (PWA)

This folder is a Progressive Web App (PWA). Open the site, then install it like an app on desktop or mobile.

## Deploy (Netlify)
1. Upload the contents of this folder as your Netlify site.
2. Make sure `index.html` is at the site root.

## Install (User)
- Open the live site in **Chrome** or **Edge**.
- Click **INSTALL APP** (bottom-right) or use the browser install icon in the address bar.

## Notes
- PWA installation requires HTTPS (Netlify provides it).
- Works on Windows/macOS/Linux/Chromebook and most Android devices.
