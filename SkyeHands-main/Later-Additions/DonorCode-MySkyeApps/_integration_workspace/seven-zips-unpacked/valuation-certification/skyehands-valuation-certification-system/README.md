# SkyeHands Valuation Certification System

Standalone platform under the SkyeHands / SkyeQuantaCore parent infrastructure.

## What is real in this build

- Accepts a codebase ZIP import into a workspace-aware runtime.
- Reconstructs descriptors, dependency graphs, launch profiles, smoke profiles, services, env templates, and runbook excerpts.
- Detects integration signals against a 2026 provider ledger.
- Produces runtime proof through static-html checks, syntax checks, launch probes, and executable smoke checks when a direct target exists.
- Executes a hermetic execution matrix across supported imported stacks, including start, smoke, test, and validate lanes when they are safely runnable.
- Scores the imported project across transparent valuation methods.
- Emits a VCS artifact family from intake dossier through audit surface, signed authority output, runtime matrix, execution matrix, repair intelligence dossier, public-trust readiness JSON, and public-trust pack ZIP.
- Builds cross-workspace portfolio scoring with portfolio certification and trend signals.
- Runs a deterministic patch lab for README, env example, license, smoke script, test script, CI workflow, Netlify config, launch manifest, validation script, and package-script repair.
- Ships a provider-ready repair subsystem with an offline repair-planning fallback and ranked file-target remediation plans.
- Ships a public-trust subsystem that generates a verified local chain, CSR, HTTP-01 and DNS-01 challenge pack, and public-issuance helper script.

## Run locally

```bash
npm start
```

Open `http://127.0.0.1:4317`.

## Smoke verification

```bash
npm run smoke
```

This command verifies the bundled proof pack and local syntax integrity shipped in this archive.

## Bundled proof pack

This archive includes a bundled proof pack under `proof-pack/`. That evidence backs the current shipped claims in this zip, including API proof for health, repair, trust, portfolio, and authority verification, plus patched workspace evidence for the sample node and polyglot fixtures.

## Platform truth rules

- This platform emits a company-side strategic certification value, not an IRS appraisal.
- Every methodology is visible.
- Every artifact is downloadable.
- Patch lab applies only deterministic supported patches and re-tests after patching.
- Updated codebases are exported in full when patching is run.
- The repair subsystem is provider-ready and smoke-proven through its offline planning fallback.
- The trust subsystem is smoke-proven through a verified local chain and public-issuance readiness pack.
