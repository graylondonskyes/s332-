GitHub scan and regression wrapper

Scan a repository, pin its baseline, compare the pull request, and export the regression proof.

This wrapper ingests a public GitHub repository URL, scans the changed files from a pull request URL, pins the current result as a baseline, compares the next result against it, and exports both the report and the regression diff.

Packaging Notes

GitHub ingestion

Paste a repository URL such as https://github.com/owner/repo or a pull request URL such as https://github.com/owner/repo/pull/123. Repository scans walk the repo tree through the GitHub contents API. Pull request scans fetch the changed files from the PR and scan those files at the PR head.
