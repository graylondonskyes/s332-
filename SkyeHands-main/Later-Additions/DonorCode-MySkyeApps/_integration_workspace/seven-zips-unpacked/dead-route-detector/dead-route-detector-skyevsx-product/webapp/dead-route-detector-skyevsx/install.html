Install and packaging flow

Use the extension, the browser scan surface, or the wrapper surface.

Editor install

  1. Package the product from the root with node scripts/package-all.mjs.
  2. Install the generated VSIX from dist/ in a VSX-compatible editor.
  3. Run Dead Route Detector: Scan Workspace.
  4. Open the report or export the JSON when the scan completes.

Browser surfaces

  1. Open scan.html for the full browser scan surface.
  2. Run the included proof fixture or select a workspace folder.
  3. Export the report JSON when needed.
  4. Use the GitHub wrapper for repo-facing packaging and quick proof review.
