What it checks

  • href, to, redirect, navigate, router.push, router.replace, and location-based path references
  • Declared routes from route objects, file-system routing, static HTML pages, and common framework conventions
  • Contributed commands, command registration, executions, menu entries, and keybindings
  • Placeholder controls such as href="#" and javascript:void(0)

What the surface does

  • Scan a chosen workspace folder directly in the browser
  • Run the included proof fixture with one button
  • Load the included sample report for review-only workflows
  • Export the currently rendered report as JSON

No theater lane

This surface is not a static brochure. The scan page contains real controls wired to the scanner, real report rendering, and real export behavior. The smoke lane clicks those controls and verifies the resulting DOM and exported payload state.
