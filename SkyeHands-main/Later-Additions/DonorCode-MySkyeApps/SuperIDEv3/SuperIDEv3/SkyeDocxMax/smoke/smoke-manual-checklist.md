# Manual Smoke Checklist

- Open `index.html`.
- Create a document.
- Save and refresh.
- Create a folder.
- Move a document into the folder.
- Export encrypted `.skye`.
- Import the `.skye` file with the correct passphrase.
- Confirm a wrong passphrase shows an error.
- Export TXT.
- Export HTML ZIP.
- Use browser print for PDF.
- Open `homepage.html` and click every visible CTA.
- Open `offline.html`.
- Disconnect network after first load and refresh the app.
- Confirm no local icon, script, CSS, or image path is missing.
