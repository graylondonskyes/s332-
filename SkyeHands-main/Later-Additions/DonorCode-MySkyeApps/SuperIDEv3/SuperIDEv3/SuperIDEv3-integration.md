# SuperIDEv3 Integration Directive

Project root: `SuperIDEv3`

Goal: combine the Dynasty SuperIDEv2 full system and the Sovereign Author Publishing System 3.3.0 live-lane pass into one new SuperIDEv3 build without losing any working product surface, backend lane, app route, proof system, or brand/polish layer.

Primary rebuild outcome: produce two deployable products from this folder:

1. `SkyeDocxMax` as a standalone document-authoring, publishing, export/import, and governance app.
2. `SuperIDEv3` as the full intelligent command system containing every preserved platform plus the embedded `SkyeDocxMax` lane.

Replacement rule: `SkyeDocxMax` supersedes `SkyeDocxPro` in the final SuperIDEv3 runtime. `SkyeDocxPro` remains a donor/source lane only until parity is proven. No final navigation label, route, manifest, smoke gate, or release note should call the final document lane `SkyeDocxPro` except when identifying the donor source.

Status rule: use ✅ only for verified present or code-backed. Use ☐ for open work. Do not mark anything complete from intention, naming, screenshots, or document-only claims.
