# 31 · POST-DIRECTIVE · SEARCHABLE ACADEMY AND TROUBLESHOOTING CANON

This pass adds a command-grade learning layer on top of the existing Academy system.

Routex and AE FLOW now each include:
- searchable help index across tours, micro guides, recipes, reports, and badge boards
- troubleshooting recipe library that launches the exact guide lane needed
- exportable learning completion reports in HTML and JSON
- role-pack completion badges with visible progress and unlock state
- topbar and in-app quick-launch controls for the new help surfaces

This pass is intended to reduce friction when the user knows the problem but does not yet know the correct feature lane.
