# INTERACTIVE_PRODUCT_EDUCATION_EXPANSION_DIRECTIVE_V36

## Intent
Turn the existing walkthrough layer into a stronger product-education system so the app teaches itself with less operator confusion.

## What this pass addresses
- expand walkthrough coverage beyond the original v35 tour set
- add first-run guided onboarding
- add contextual help launchers on deep surfaces
- add an in-app coverage matrix so the product can prove what is already taught and what still needs to be run
- keep the tutorial system embedded in the shipped product instead of relying on outside docs
