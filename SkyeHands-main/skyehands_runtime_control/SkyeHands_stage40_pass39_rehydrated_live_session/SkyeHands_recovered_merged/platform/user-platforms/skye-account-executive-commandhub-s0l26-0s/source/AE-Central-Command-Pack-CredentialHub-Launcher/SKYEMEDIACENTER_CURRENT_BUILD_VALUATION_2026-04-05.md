## Why

This build now adds a real artist-facing media surface into the platform:
- shareable public artist pages
- searchable releases
- direct in-browser audio and image uploads
- photo-drop publishing
- pre-release dates
- direct video upload
- server-side poster generation
- multi-variant MP4 transcode outputs
- public stream playback from artist pages and feed surfaces
- IndexedDB local preservation
- Neon-backed metadata sync plus SQL video-asset ledgers

## Current boundary

This pass closes the old embed-first video boundary. Embeds still remain available as a fallback lane, but direct video upload, transcode, and public playback now exist in the product.
