Core infrastructure layers

  • SkyeHands Platform Bus: File-backed signed event queue — SHA-256 envelope hashing, nonce-based replay protection, NDJSON immutable audit ledger. Routes 18 event types across 8 platform subscriptions. Transport swappable to Cloudflare Queue or Neon outbox without touching business logic.
  • AE Brain Mesh: 13 specialized Netlify Functions agent brains (onboarding, growth, commerce, appointment, media, support, legal, finance, tech, deployment, music, lead, skydexia). Each has role-scoped permissions, Anthropic → OpenAI → Gemini fallback chain, max-hop loop protection, memory fabric, usage tracking, and audit trail.
  • AE Command Hub: Multi-tenant-first PostgreSQL/Neon schema — Tenants, Users, Sessions, AE Brains, Clients, Leads, Tasks, Threads, Messages, Appointments, Billing Events, Audit Log. RBAC: founder, admin, ae_operator, client, viewer. All tables tenant-scoped from the ground up.
  • Provider Contract System: 13 explicit provider contracts (OpenAI, Anthropic, Gemini, Stripe, Printful, Twilio, Resend, Calendly, Google Calendar, Microsoft 365, GitHub, Cloudflare, Neon). Single dry-run dispatch function per provider — no separate code paths. Fails loud on missing env vars.
  • SkyeQuantaCore Shell: Python + Node.js workspace orchestrator — workspace scheduler, NDJSON runtime journal, autonomous dependency repair, multi-workspace state management (local-default, remote-default, preview-stage).
  • kAIxUGateway13: Full-stack governed AI routing platform — Netlify Functions, Stripe billing, Neon, Redis, device key management, cohort analytics, 40+ serverless functions. Powers all 9 SkyeSuite apps.
  • SkyDexia AI Brain Drive: 90GB Docker/GPU platform for RAG, model fine-tuning, and Memory Fabric v6.8.1 — Node.js distributed memory layer for agent persistence.
  • Proof Discipline / GrayChunks: Claims → smoke scripts → proof files → GrayChunks scanner (13 violation rules) → directive release gate. All status docs are code-backed. No aspirational claims.
