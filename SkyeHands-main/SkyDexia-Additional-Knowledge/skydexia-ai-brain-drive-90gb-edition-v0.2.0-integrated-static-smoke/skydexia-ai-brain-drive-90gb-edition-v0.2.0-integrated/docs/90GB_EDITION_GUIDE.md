# SkyeDexia 90GB Edition Guide

This edition makes the portable AI drive concept realistic on a roughly 90GB drive.

Recommended default:
- qwen2.5-coder:7b
- phi4-mini
- llama3.2:3b

Optional only if at least 35GB remains before pull:
- deepseek-coder-v2:lite

Do not default these on 90GB:
- 32B model packs
- 70B model packs
- frontier-class full weights
- multi-hundred-B parameter archives
