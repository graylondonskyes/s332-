SkyeDexia Lite Drive — 90GB Edition

A compact portable AI workspace for local coding, document synopsis, extraction, and business workflow support.

SkyeDexia Lite Drive gives users a local AI workspace that can run practical open-weight models from an external drive. It includes SkyeDexia orchestration, a branded local interface, model routing, setup scripts, prompt vaults, hardware checks, and offline documentation.

The drive stores the AI package and model files. The customer's CPU/GPU/RAM provides the compute.
