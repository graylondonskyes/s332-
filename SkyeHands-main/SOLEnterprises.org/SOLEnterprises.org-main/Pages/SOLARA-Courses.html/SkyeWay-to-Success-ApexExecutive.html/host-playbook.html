Partner Value (Use the same words sold in the pitch)

This program is positioned as: “not templated files” with a detailed intake process, a Week 3 polished website, and a Week 6 press package — plus a white glove filing pipeline with third-party review before submission.

  • Host role: operational excellence so the program runs like a premium executive environment.
  • Host does not teach: host provides facility, check-in/out, and incident integrity.
  • Confidentiality posture: enforce “no recording / no live streaming.”

Host is not granted curriculum access. If a host POC is present, they sign NDA and agree not to record or distribute materials.

Host DOES / DOES NOT

  • Host DOES: facility access, check-in/out, safety, emergency procedures, incident documentation, enforce “no recording.”
  • Host DOES NOT: teach/co-teach, request curriculum, record sessions, distribute materials, promise outcomes.

Facility Quality Standard

  • Clean, professional environment.
  • Controlled entry + predictable room assignment.
  • Reliable seating/lighting + basic privacy.
  • Signage supported: “NO RECORDING.”

Pre-Session Setup

  • T-60: unlock room, seating, HVAC/lights, exits clear.
  • T-15: check-in station ready, roster ready, incident form ready, signage posted.
  • Privacy: session area controlled; avoid unnecessary foot traffic.

Check-In Protocol (Adults)

  • Verify roster match (name match; optional ID per facility policy).
  • Mark arrival time + notes (late, guest attempt, behavioral flags).
  • Late: quietly within 15 minutes. After that, notify CS/PD before admitting.
  • Guests: only if PD pre-approved (NDA signed at check-in if applicable).

Conduct Enforcement

  • No recording / live streaming.
  • No disruptive behavior.
  • No unauthorized access to instructor devices.
  • Escalation: warning → notify CS/PD → PD decides removal → host supports safe exit.

Incident Response

  • Categories: Medical • Behavioral • Facility/Security • Tech/Safety.
  • Medical: call 911 if urgent; follow facility procedure; notify PD/CS; document facts/times/witnesses.
  • Behavioral/Security: do not escalate; separate parties if needed; notify PD/CS; call security/911 if threatening; document facts only.
  • Post-incident: host sends incident report to PD/CS within 2 hours same day.

Close Protocol

  • Mark session end; reset room; ensure no materials left behind.
  • Lock up; confirm next-session readiness.

Communication Ladder

  • Day-of ops: Host → CS
  • Escalations: Host → PD
  • Emergency: 911/facility security first → then PD
